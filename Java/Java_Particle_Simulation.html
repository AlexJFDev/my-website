    This is a Java program that simulatates a lot of particles simultainiously.
    The particles bounce off of the wall and each other and a hashtable is used for collisions.
    It isn't perfect though and still needs some work.

    If the file doesn't work due to "unidentified developer" on a macbook you can run:

    xattr -c java-particle-simulatior.jar

    and it will be capable of running.

    Lastly it may not work properly if you are not up to date with Java version 17.

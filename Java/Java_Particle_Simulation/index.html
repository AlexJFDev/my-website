This is a Java program that simulates a lot of particles simultaneously.
The particles bounce off of the wall and each other and a hash table is used for collisions.
It isn't perfect though and still needs some work.

If the file doesn't work due to "unidentified developer" on a MacOS you can run:

xattr -c java-physics-simulator.jar

and it will be capable of running.

Lastly it may not work properly if you are not up to date with Java version 17.

You can now load and save the state of the simulation with serialized files. I've given a sample file called particleRow.ser to try. ".ser" is the standard ending for serialized Java files but you can use any ending you'd like or none at all.

This is a Java program that plays midi sounds from a JSON file.

Test Song

Start the program with the following command:

java -jar java-midi-player.jar

Alternatively you can just double click to run it.
Keep in mind the program will need read permissions to work properly.
Then click "Select a File", next select your song file, and then click open.
The song should begin to play.
